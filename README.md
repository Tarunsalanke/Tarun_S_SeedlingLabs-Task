An AI-powered web application that analyzes GitHub issues and generates a structured summary to help engineering teams quickly understand and prioritize incoming issues.

This project was built as part of the **Seedling Labs Engineering Intern Craft Case**, focusing on agentic thinking, clean system design, and practical AI integration.

---

## ğŸš€ Features

- Accepts any **public GitHub repository URL**
- Analyzes a specific **GitHub issue number**
- Fetches issue title, description, and comments using GitHub API
- Uses **Groq LLM (llama-3.1-8b-instant)** to generate structured insights
- Classifies issue type (bug, feature request, documentation, etc.)
- Assigns priority with justification
- Suggests relevant GitHub labels
- Clean and simple **Streamlit UI**
- Copy-ready structured JSON output
- **Download the JSON Response**: You can easily download the structured JSON output generated by the application for further analysis or integration into your workflow. Simply click the download button provided in the Streamlit UI after processing an issue.
